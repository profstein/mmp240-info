# MMP 240 Course Hub

Static multi-page hub for the MMP 240 Web Design class.

## Structure

```
index.html          Home landing page
assignments/        Assignments for the class
weeks/              Weekly outline 
projects/           Project listings 
topics/             Topical references 
help/               Help & resources 
```

## Tech & Constraints

- Pure semantic HTML5, modern responsive CSS, and vanilla JS where possible, lightweight libraries otherwise.
- Accessible navigation (skip link, focus styles, ARIA states).
- No build tools
