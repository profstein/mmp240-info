Project 3: Responsive Web Site

Overall, this project is to take Project 2 and redesign it to make it fully responsive, improve visual design, and expand it into a multi-page site featuring individual landmark pages.

In order to do that we will:

  • Add additional landmarks and pages for each landmark
  • Add a responsive navigation to the header so that we can navigate to pages.
  • Add media queries to get more complete control over how it looks across devices

We will also:

  • learn how to work with AI, and use this to help write the landmark pages and other parts.
  • revisit some past skills
  • learn a new way to publish our sites.

Requirements

  • Responsive Design

    • [new] Media Queries Add media queries (at least 2 breakpoints: mobile + desktop; tablet optional) so they layout changes at the breakpoint(s).
    • [from project 2] Flexible Media You have the CSS in place and HTML structure so that images and video can scale and fit within their containers.
    • [from project 2] Fluid Grid Your layout is in fluid units (fr, %) so that content can adapt to screen size.

  • Navigation

    • A responsive navigation that links to the home page and all six landmark pages.

  • Multi-Page Structure

    • Create individual landmark pages: 6 — one per landmark.
    • Each landmark page must follow the same structure:
      • h1 landmark name
      • Hero image (use the one provided). Bonus for using an art-directed picture element.
      • The text provided in the markdown files
      • Previous / Next buttons (these should lead you through all 6 pages)
    • All pages on the site have the same header (including navigation) and footerReuse the same header, nav, and footer across all pages.

  • Visual Design

    • Typography is consistent and follows a typographic hierarchy
    • Colors are harmonious and consistently used, including colors to represent links and buttons
    • Layout and spacing allow for differentiation of page sections and content and easy scanning.

  • Accessibiity

    • All images should have alt attributes
    • Text and background colors have sufficient contrast
    • Semantic HTML elements are used
