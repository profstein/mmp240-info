# Responsive Web Design Resources

<h2>What Is Responsive Web Design</h2>
<ul style="list-style-type: circle;">
<li>Ethan Marcotte, “Responsive Web Design.” <a rel="noopener" href="https://alistapart.com/article/responsive-web-design/">A List Apart</a></li>
</ul>
<p style="text-align: center;">The original 2010 article that introduced RWD and the ideas of fluid grids, flexible images, and media queries.</p>
<ul style="list-style-type: circle;">
<li>Introduction to Responsive Design. <a rel="noopener" href="https://web.dev/learn/design/intro">web.dev/learn/design/intro</a></li>
</ul>
<p style="text-align: center;">Overview of responsive design history, principles, and why it matters for users.</p>
<ul style="list-style-type: circle;">
<li>Responsive Design — MDN Web Docs. <a rel="noopener" href="https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/CSS_layout/Responsive_Design">developer.mozilla.org</a></li>
</ul>
<p style="text-align: center;">A foundational tutorial from Mozilla covering layout techniques, viewport meta tags, and responsive best practices.</p>

<h2>Fluid Grid Layouts</h2>
<ul style="list-style-type: circle;">
<li>Macro Layouts. <a rel="noopener" href="https://web.dev/learn/design/macro-layouts">web.dev/learn/design/macro-layouts</a></li>
</ul>
<p style="text-align: center;">Explains large-scale page structure using flexible grid systems and fractional (fr) units.</p>
<ul style="list-style-type: circle;">
<li>Micro Layouts. <a rel="noopener" href="https://web.dev/learn/design/micro-layouts">web.dev/learn/design/micro-layouts</a></li>
</ul>
<p style="text-align: center;">Focuses on smaller-scale layout details such as cards and component alignment inside a flexible grid.</p>

<h2>Media Queries and Breakpoints</h2>
<ul style="list-style-type: circle;">
<li>Media Queries — MDN Web Docs. <a rel="noopener" href="https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/CSS_layout/Media_queries" target="_blank">developer.mozilla.org</a></li>
</ul>
<p style="text-align: center;">Introduces the @media rule and syntax for targeting different screen sizes and device features.</p>
<ul style="list-style-type: circle;">
<li>Media Queries Guide. <a rel="noopener" href="https://web.dev/learn/design/media-queries" target="_blank">web.dev/learn/design/media-queries</a></li>
</ul>
<p style="text-align: center;">Covers the mobile-first approach and performance considerations for responsive CSS.</p>
<ul style="list-style-type: circle;">
<li>Choosing Breakpoints in Responsive Design. <a rel="noopener" href="https://www.freecodecamp.org/news/breakpoints-for-responsive-web-design" target="_blank">freeCodeCamp.org</a></li>
</ul>
<p style="text-align: center;">Explains common breakpoint ranges and how to select meaningful ones for your layout.</p>
