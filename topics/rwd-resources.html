Responsive Web Design Resources

What Is Responsive Web Design

  • Ethan Marcotte, "Responsive Web Design." A List Apart

The original 2010 article that introduced RWD and the ideas of fluid grids, flexible images, and media queries.

  • Introduction to Responsive Design. web.dev/learn/design/intro

Overview of responsive design history, principles, and why it matters for users.

  • Responsive Design — MDN Web Docs. developer.mozilla.org

A foundational tutorial from Mozilla covering layout techniques, viewport meta tags, and responsive best practices.

Fluid Grid Layouts

  • Macro Layouts. web.dev/learn/design/macro-layouts

Explains large-scale page structure using flexible grid systems and fractional (fr) units.

  • Micro Layouts. web.dev/learn/design/micro-layouts

Focuses on smaller-scale layout details such as cards and component alignment inside a flexible grid.

Flexible Media

  • Responsive Image Example (CodePen). codepen.io/profstein/pen/mqMBoV

Demonstrates how to make <img> elements scale within their containers.

  • Responsive Video Embed Example (CodePen). codepen.io/profstein/pen/JoGLoKX

Shows the "wrapping container" method for embedding responsive YouTube or Vimeo videos.

Media Queries and Breakpoints

  • Media Queries — MDN Web Docs. developer.mozilla.org

Introduces the @media rule and syntax for targeting different screen sizes and device features.

  • Media Queries Guide. web.dev/learn/design/media-queries

Covers the mobile-first approach and performance considerations for responsive CSS.

  • Choosing Breakpoints in Responsive Design. freeCodeCamp.org

Explains common breakpoint ranges and how to select meaningful ones for your layout.
