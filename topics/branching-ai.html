Example: Combining AI and Branching

Intro

The video below shows an example of creating a new branch, using Copilot to assist with code changes, and then pushing and merging the branch back into main.

If you are behind on Project 2, note that the example demonstrates wrapping <figure> elements in containers and adding <h3> headings driven by the alt text. Watch the segment titled Turning Landmarks into Cards before you start working.

Video Walkthrough

The narrator shows how to: create and switch to a branch, use Copilot to generate the HTML structure, review the AI suggestions, stage and commit the updates, and finally merge the branch back into main.
