Copilot Setup to use AI in VS Code

Intro

This guide walks through how to get Copilot set up in VS Code. Copilot is an overall AI assistant and framework in Visual Studio Code. It allows you to choose from a few different models. If you set it up with a verified education account, you will get free use of it with a certain number of credits each month. Without an educational account enabled, you would have to pay for it.

Video Walk through

I missed a final step in the above video when I was showing the example of how to use AI to create the HTML for the cards we made in Project 2. This shows what you need to click to accept the code that AI has written.

Steps to get Copilot

  1. Check if you have education benefits.
    1. Sign in to Github
    2. Go to this page: https://github.com/settings/education/benefits
    3. If you don't have them, click Start and Application
  2. Once you have education Benefits. Sign Up for CoPilot
    1. https://github.com/github-copilot/free_signup
  3. Setup Copilot in VS Code: https://code.visualstudio.com/docs/copilot/setup

Resources and Further Learning

  1. Getting Started: https://code.visualstudio.com/docs/copilot/getting-started
  2. Use GitHub Copilot to get code suggestions in your editor: https://docs.github.com/en/copilot/how-tos/get-code-suggestions/get-ide-code-suggestions?tool=vscode
  3. Prompt Engineering for Copilot Chat: https://code.visualstudio.com/docs/copilot/chat/prompt-crafting
    1. YouTube Video from above page: Master the core principles of prompt engineering with GitHub Copilot
  4. Learn how to use Copilot commit message generation responsibly by understanding its purposes, capabilities, and limitations: https://docs.github.com/en/copilot/responsible-use/copilot-commit-message-generation
  5. More about Copilot in VS Code: https://docs.github.com/en/copilot/concepts/completions/code-suggestions?tool=vscode
