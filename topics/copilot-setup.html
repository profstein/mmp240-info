Copilot Setup to use AI in VS Code

Intro

This guide walks through how to get Copilot set up in VS Code. Copilot is an AI assistant baked into Visual Studio Code, with multiple model choices. When you connect a verified education account you get free monthly credits; otherwise you would pay for access.

Video Walkthrough

This video shows the whole setup, including the moment when you accept AI-generated code inside the editor.

Steps to Get Copilot

  1. Check if you already have education benefits:

    1. Sign in to GitHub.
    2. Visit https://github.com/settings/education/benefits.
    3. If you do not have the benefits yet, click Start an Application.

  2. Once you do, sign up for GitHub Copilot at https://github.com/github-copilot/free_signup.

  3. Install Copilot in VS Code by following https://code.visualstudio.com/docs/copilot/setup.
