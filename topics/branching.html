Pushing Changes from your computer to GitHub and Merging Changes back to the main Branch

After you have made all of your changes for your branch and committed them, you can push that to GitHub and merge the branch into the main branch.

Opening the Terminal in VS Code

All of the code below is terminal commands. You already have access to the terminal in VS Code; here is how to open it:

  1. With a project open, look at the bottom of the VS Code window. If you see a panel with the tabs Problems | Output | Debug Console | Terminal, click Terminal.

    1. If you do not see that panel, click the toggle panel button (or press ⌘ + J), then click Terminal:

  2. If the terminal already has something running (like Live Server), open a new shell. Click the caret next to the + icon and choose Bash.

Now you should be ready to enter commands in the terminal.

1. Check Your Current Branch

git status
git branch

  • git status shows what is happening in your working directory.

  • git branch lists all local branches.

  • The branch with an asterisk (*) is the one you are currently on—usually main.

2. Create a New Branch

This creates a new branch called feature-images based on your current branch (often main).

git branch feature-images

Tip: use descriptive names such as fix-navbar, add-gallery, or update-css-layout.

3. Switch to That Branch

git switch feature-images

You can also use the older command git checkout feature-images.

Now you are working in the new branch—anything you commit stays isolated from main.

You can combine steps 2 and 3 by adding the -c flag: 

git switch -c feature-images

4. Make Changes and Commit

Edit files and then run:

git add .

This stages your edits.

git commit -m "Added responsive image section"

5. Push the Branch to GitHub

git push -u origin feature-images

The -u flag connects your local branch to the same branch on GitHub.

Merging a Branch into Main

1. Switch Back to Main

git switch main

This is the code you always use to return to the main branch.

2. Merge the Branch into Main

git merge feature-images

Concept Summary

Command Purpose
git branch List or create branches
git switch Move between branches
git merge Combine branch changes
git add Adds changes to staging
git commit Commits your changes
git push / pull Sync local and remote branches
