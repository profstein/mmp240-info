Welcome to MMP 240

This course explores the foundations of web design, from HTML and CSS to responsive layouts and version control with Git and GitHub.

Projects

Major assignments. There are four of them in the course.

Project 2: Responsive Media Page

Due: October 20

Build a fluid grid with responsive images and video, applying flexible media techniques and visual design principles.

Project 3: Responsive Web Site

Due: November 19

Expand your project into a multi-page responsive site with navigation, media queries, and individual landmark pages.
